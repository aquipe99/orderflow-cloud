# OrderFlow Cloud

Plataforma distribuida de procesamiento de órdenes basada en microservicios y arquitectura orientada a eventos.

## Arquitectura

Tecnologías principales:

- Java 17
- Spring Boot
- Spring Cloud Gateway
- Kafka
- Avro
- PostgreSQL
- Quarkus
- Docker
- Kubernetes
- Azure AKS

## Microservicios

- API Gateway
- Order Service
- Inventory Service
- Payment Service
- Notification Service

## Objetivo

Proyecto desarrollado para demostrar conocimientos en:

- Microservicios
- Arquitectura Event Driven
- Cloud Native
- CI/CD
- Observabilidad
